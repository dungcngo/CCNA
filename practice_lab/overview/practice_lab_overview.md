## Cisco CCNA Practice Lab Overview

The Cisco CCNA Lab is based on the simplest topology needed to achieve the objectives of the exam blueprint.

![lab topology](overview/lab_topology.jpeg)

The full equipment list is detailed below:
### Site 1 (HQ, 200 users)
- Two Cisco IOS-XE routers
- Three Catalyst Layer 3 switches
- One Catalyst Layer 2 switch
- One Linux-based server
- Two workstations

### Site 2 (SOHO office, 2 users)
- One Cisco IOS-XE router
- One Catalyst Layer 3 switch
- One workstation
- One IP phone

### Site 3 (Branch office, 50 users)
- One Cisco IOS-XE router
- One Catalyst Layer 3 switch
- Two Catalyst Layer 2 switches
- Thre workstations
- One Wireless LAN Controller
- One Lighwight Access Point
